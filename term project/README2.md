# 기말 프로젝트 2차 발표



## 게임 소개

스타크래프트 유즈맵인 포커디펜스를 따라한 게임

포커를 통해 기물을 얻어 디펜스를 진행

## 현재까지의 진행상황

### 구현해야할 항목들과 진행 상황

1. 게임 스테이트를 만들고 각 스테이트간의 이동을 구현 -> 랭킹보드빼고 구현 (75% 완성)

2. 포커를 진행해서 기물을 얻는다 -> 포커알고리즘 반복 수정중에 있음(25%)

3. 웨이브와 스테이지 구현해서 적들이 나오는 것 -> 미구현(0%)

4. 기물을 얻고 배치하고, 기물들이 적을 공격하는 애니메이션 -> 미구현(0%)

### 커밋 목록

이번 주말에 몰아서 올렸기에 커밋목록은 따로 올리지 않겠습니다.

### 목표가 변경되었다면 변경된 내용과 이유

아직 목표는 변경되지 않았지만 계획에 차질이 생겼습니다. 
그래도 포기하지 않고 해볼 생각입니다.

### 구현하면서 어려웠던 부분이나 추가로 다뤘으면 하는 부분

아직 좀더 다뤄봐야 알것같다.
